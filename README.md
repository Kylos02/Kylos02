### 🌟 Proyectos destacados

#### ⚙️ [GestorDeProcesos](https://github.com/Kylos02/GestorDeProcesos)
> Simulador de Sistema Operativo desarrollado para el curso de S.O.
- Algoritmos de **scheduling de CPU**, gestión de memoria y ciclo de vida de procesos
- Construido con **Java**

#### 🌐 Web Trend Extractor
> Extracción automatizada de tendencias en tiempo real desde plataformas digitales
- Web scraping con **Python** + **BeautifulSoup**
- Datos estructurados y listos para análisis

#### 🗄️ Restaurant Management System
> Arquitectura relacional completa para gestión de restaurante
- Diseñado con **PostgreSQL**
- Control de inventario y reportes operacionales con queries avanzados

#### 🎵 [Music Player](https://github.com/Kylos02/MusicPlayer)

- **Raspberry Pi Zero 2W** + pantalla + DAC PCM5102A
- Compatible con **Spotify**, construido con **Python** 🍓
